# Mobile-First UX Strategy & PWA Guidelines

## Mobile-First Priority Workflows
1. **Housekeeping Dispatch & Room Status Update**: High-touch mobile PWA interface for room attendants updating cleaning status live on mobile devices.
2. **Guest Mobile Key & Check-In Portal**: Responsive web storefront view providing mobile check-in and active TTLock door passcode display.
3. **Manager On-the-Go Approvals**: Mobile-optimized dashboard for General Managers to approve rate overrides and refund requests.

## Responsive Layout Breakpoints
- **Mobile (`< 640px`)**: Single-column layout, bottom navigation bar, touch-friendly 48px hit targets.
- **Tablet (`640px - 1024px`)**: Collapsible sidebar, 2-column KPI grids.
- **Desktop (`> 1024px`)**: Persistent multi-level sidebar, full tape chart Gantt grid, side-by-side ledger screens.
